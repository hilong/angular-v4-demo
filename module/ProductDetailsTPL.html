购买须知

买车须知

  1. 汽车属特殊商品，建议您购买或支付前务必与客服人员进行沟通，并确认配置、价格、库存、提车时间及地址。
  2. 您在支付以后需要完善购车真实信息，包括姓名，手机号码，身份证号码，地址等，方便客服人员及时与您联系。
  3. 在您签订《客户交车确认表》前，您有权利检验待交车辆，如果有任何不满意，可以随时要求取消或者修改订单，取消订单订金将无条件退还。
  4. 由于汽车属特殊商品，签订《客户交车确认表》后，将无法享受无理由退换货服务，所售车辆将严格按照国家相关法律及汽车生产厂家的质保政策执行。

关于付款

  1. 我们为您提供线上支付订金(线下付尾款)、分期付款(线上付订金，线下付首付)两种支付方式。
  2. 我们支持使用多张借记卡或信用卡将订单金额进行多笔支付，但每张卡仅能支付一笔。
  3. 线上订金支付需在2个小时内完成，且支付金额需达到2000元，我们即可帮您保留车源，如2小时内未完成我们会将订单释放并不再为您保留车源，您之前支付 的款项我们将通过银联原路退还给您。
  4. 分期付款在线上支付订金并上传所需资料后，会有金融专员跟进您的分期手续进度，待分期批复后通知您办理提车手续，提车时支付首付款。

关于提车

  1. 在您在线支付订金完成后，我们会有客服联系您确定提车时间，届时请保持电话畅通。
  2. 我们为您将车辆保留在自提点7天(自然日)，超过7天时间我们将不予保留该车，订金会自动退回您的支付账户。
  3. 提车时需本人持身份证。
  4. 汽车需客户当面验收，签订《客户交车确认表》后代表将车辆的所有权正式递交给您。
  5. 签订《客户交车确认表》后，将有专人安排您车辆的用品安装，保险、购置税购买，车辆上牌。

常见问题

提问：车源质量是否有保障？
回答：恒信汽车电商承诺所售车辆均为全新车，并按国家及汽车生产厂家的三包标准享受全国联保。
提问：全额支付后是否可以退车？
回答：签订《客户交车确认表》前，可随时申请退车，并全额退还车款至您付款的账户。签订《客户交车确认表》后，符合国家质量监督检验检疫总局《家用汽车产品修理、更换、退货责任规定》中情况的可以进行退车。
提问：支付是否安全？
回答： 恒信汽车商城提供具有安全保障的支付渠道，充分保障您的支付渠道及敏感信息的安全，如发生资金被盗，恒信汽车电商承诺100%赔付。
提问：是否现车？
回答： 标注“现车”的车型有现车，但可能因为现车所在地不在您所在的城市而存在3-7天的物流期。如标注“需预定”，则表示无现车需要预定，预定的时间需提前联系客服确认。
提问：为何恒信汽车电商的车价远低于市场平均行情？
回答： 恒信汽车是全国领先的汽车经销集团，拥有出色的整车供应链体系，现创新开展汽车电商模式，旨在用更低的成本与更高效的经营方式颠覆传统汽车销售方式，在传统4S店汽车销售价格之上省去了4S店场地费用、广告费、库存资金利息、销售员奖金等费用，将价格一降到底让利于消费者。
提问：是否提供代办购置税、保险、上牌服务？
回答： 我们提供代买购置税、保险、上牌代办的服务，服务费500元（含上牌成本费用）。
提问：网上购车后如何提车？
回答： 提车需要到提车点自提，提车点地址：武汉市汉阳区龙阳大道邱家大湾特1号（武汉星凯4S店内）。
提问：是否可以办理车贷？
回答： 可以办理车贷，您在下订单的时候选择“分期付款”，然后在线上订金支付完成后，上传办理分期所需的资料，我们会有专业人员协助您办理车贷。
提问：上牌手续是否齐全？
回答： 我们所供车源全部来源于正常渠道，均能保证上牌所需手续齐全（国产车：发票、合格证；进口车：发票、进口机动车随车检验单、车辆购置税电子证书、车辆一致性证书、货物进口证明书）。 备注：部分地区需自行到相关部门办理地区上牌所需的特别证件（如：环保证明等）。
提问：上牌的区域？
回答： 我们所有车辆面向全国客户销售，对上牌区域没有限制，您可以凭车辆相关上牌手续到任意地区上牌（有限购政策的区域需您自行获取上牌资格）。
